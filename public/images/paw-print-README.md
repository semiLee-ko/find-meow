# 발자국 이미지 안내

## 필요한 파일
`paw-print.png` - 고양이 발자국 이미지

## 위치
이 파일을 `images` 폴더에 넣어주세요:
```
find-meow/
  └── images/
      └── paw-print.png
```

## 권장 사양
- **크기**: 100x100px ~ 200x200px (정사각형)
- **배경**: 투명 (PNG)
- **색상**: 회색 또는 검은색 계열 (배경에 잘 보이도록)
- **스타일**: 귀여운 고양이 발자국 모양

## 무료 아이콘 사이트
- Flaticon (flaticon.com)
- Icons8 (icons8.com)
- Freepik (freepik.com)

검색어: "cat paw print", "paw icon", "animal footprint"
