# Find Meow 🐱 - TV 채널 고양이 찾기 게임

모바일 웹뷰(Capacitor WebView)에서 동작하는 귀여운 TV 채널 선택 미니게임입니다.

## 📁 프로젝트 구조

```
find-meow/
├── index.html              # 메인 게임 파일 (HTML + CSS + JS 통합)
├── images/
│   └── channels/          # 채널 이미지 폴더
│       ├── channel1.png   # 고양이 이미지 1
│       ├── channel2.png   # 고양이 이미지 2
│       └── ...            # channel10.png까지
└── point/
    └── json/
        └── pointInfo.json # 각 이미지의 고양이 위치 좌표 정보
```

## 🎮 게임 방법

1. **참여 인원 선택**: 1~10명까지 선택 가능
2. **게임 시작**: 리모컨 UI가 나타납니다
3. **각 플레이어 차례**:
   - 이름 입력
   - 숫자 버튼(0~9)으로 채널 번호 입력 (1~999)
   - "채널변경" 버튼으로 랜덤 고양이 이미지 표시
   - TV 화면에 고양이 위치 포인터가 표시됩니다
   - "확정" 버튼으로 다음 플레이어로 넘어가기
4. **결과 확인**: 고양이를 가장 많이 찾은 순서대로 순위 표시

## 🖼️ 이미지 추가 방법

1. `images/channels/` 폴더에 PNG 이미지 파일을 추가합니다
   - **파일명 형식**: `channel0001.png`, `channel0002.png`, ... `channel0100.png`
   - 숫자는 4자리로 왼쪽에 0을 채워서 저장합니다
   - 예: 1번 → `channel0001.png`, 25번 → `channel0025.png`

2. 코드에서 이미지 개수 조정 (선택사항)
   - `index.html` 파일을 열어 이미지 생성 루프의 최대값을 수정합니다:
   ```javascript
   // 기본값: 100개 (channel0001.png ~ channel0100.png)
   for (let i = 1; i <= 100; i++) {
       // 실제 이미지 개수에 맞게 100을 변경하세요
   }
   ```

3. **중복 방지 기능**
   - 한 게임 내에서 같은 이미지가 두 번 나오지 않습니다
   - 각 플레이어가 채널을 변경할 때마다 아직 선택되지 않은 이미지 중에서 랜덤으로 선택됩니다
   - 참여 인원이 많을 경우, 충분한 이미지를 준비해주세요

## 📍 고양이 위치 정보 설정

`point/json/pointInfo.json` 파일에 각 이미지의 고양이 위치를 설정합니다:

```json
{
  "channel0001.png": {
    "cats": [
      {
        "id": 1,
        "x": 0.343203,  // X 좌표 (0.0 ~ 1.0, 왼쪽에서 오른쪽)
        "y": 0.172629   // Y 좌표 (0.0 ~ 1.0, 위에서 아래)
      }
    ]
  }
}
```

- **x, y 값**: 0.0 ~ 1.0 사이의 비율 값 (0.5 = 중앙)
- 예: x=0.25는 왼쪽에서 25% 위치, y=0.75는 위에서 75% 위치

## 🚀 실행 방법

### 로컬 테스트
1. 간단한 HTTP 서버 실행:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (http-server 설치 필요)
   npx http-server -p 8000
   ```

2. 브라우저에서 `http://localhost:8000` 접속

### Capacitor WebView 배포
1. Capacitor 프로젝트에 파일 복사
2. 빌드 및 배포
